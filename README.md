# webhook_deploy

### 克隆存储库
> 引用
```git clone git@github.com:henlei153/webhook_deploy.git```  
```cd webhook_deploy```

### 定义你的config.ini
```cat config.ini``` 
```> 例：  
[git@github:hanlei153/webhook_deploy.git]  
;path为代码部署目录  
path=/home/www```

### 部署
```pip3 install flask```  
```python3 webhooks.py```
